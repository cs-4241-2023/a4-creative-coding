Black Hole Audio Visualizer


This is a 2d Audio Visualizer that uses a predetermined mp3 file as a proof of concept of the abilities of the visualiser. This application has 5 points of interest to explore.


  1. The Play button is used to initially start the visualization and can also be used to pause the application once it is running.
  2. The Brightness slider will change the intensity of the light escapeing from the visualiser.
  3. The Bar Size slider will change the length of the bars within the visualiser.
  4. The Circle Size slider will change the radius of the visualiser.
  5. The Line Width slider will change the width of the bars protruding from the visualiser.
