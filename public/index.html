Description

    Originally created by John Conway (1970)

    The Game of Life is a zero-player game. The result is determined by the initial state, though edits can be made by a user while the game runs.

    The game includes a grid of squares. Each square can be in one of two states: alive or dead. Following the rules of the game, the squares 'evolve' each turn (Evolution Time).

    Here are the Rules for The Game of Life:

    1. Any live square with fewer than two live neighbors dies (underpopulation).
    2. Any live square with two or three live neighbors survives to the next generation.
    3. Any live square with more than three live neighbors dies (overpopulation).
    4. Any dead square with exactly three live neighbors becomes a live square (reproduction).

    The rules can lead to interesting patterns. The game has created patterns of real world patterns such as oscillators and spaceships.

Controls

There are 5 Different Features of the Game the user can control:

  1. What squares are alive and what squares are dead (control by clicking on squares)
  2. Pause or Play the Game (button)
  3. Change the color of the squares (Pick a Square Color) or Use Random Colors (checkbox)
  4. Change to Board Size (slider on top)
  5. Change to Evolution Time (slider on bottom)
