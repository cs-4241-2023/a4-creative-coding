Conway's Game of Life

Author: Bright Lin

To start the game with your own board, use your mouse to click on which cells you want to be alive.

To start the game with a random board press the Random Board button.

You will be able to pause the game with the Pause button and resume it with the Start Button.

Using the Speed Slider you will be able to change the speed of the game.

There is a pre-set board called Glider that will result in the Glider formation.

Finally you can clear the board with the Clear Board button.
