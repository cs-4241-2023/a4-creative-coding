Visualization Appearance Control Parameters

Although it is recommended to go through the following steps for controlling the visualization appearance in the presented order, moving through the steps in any desired order is also acceptable. Furthermore, a user can set the parameters below before activating the audio visualization and also during the visual experience.

Step 1: Use the dropdown selection below to set the visualization drawing to generate occurrences of a listed shape. The two options currently available are rectangles and circles. By default, the visualization drawing generates occurrences of a rectangle.

Step 2: The following fields should be used to change the fill color of the visualization rectangle or circle drawings using the RGB scale. The default fill color of the drawings is black (RGB(0,0,0)):

Step 3: The following fields should be used to change the stroke color of the visualization rectangle or circle drawings using the RGB scale. The default stroke color of the drawings is black (RGB(0,0,0)):

Step 4: The following fields should be used to change the appearance of the shadow of the visualization rectangle or circle drawings. Enter information for all fields below, which include the shadow color, shadow blur, shadow offset along the x-axis, and shadow offset along the y-axis:

Step 4.a: Change the shadow offset along the x-axis (horizontal) and y-axis (vertical). Both offsets are set to 0 by default:

Step 4.b: Change the shadow color using the RGB scale. The shadow color is set to black (RGB(0,0,0)) by default:

Step 4.c: Change the amount of shadow blur. The shadow blur value is set to 0 by default, and an increase in value along the slider corresponds to a greater degree of shadow blur:
